# Student Assessment Data Analysis

## Objective
Analyze student performance data by consolidating, cleaning, and generating insights using Excel.

## Key Tasks
- Data consolidation from multiple sheets
- Data cleaning (duplicates, missing values)
- Data standardization
- Dashboard creation using pivot tables and charts

## Tools Used
- Microsoft Excel

## Insights
- Identified top-performing zones and cities
- Found correlation between attendance and performance
- Highlighted subject-wise strengths and weaknesses
